# Text Analyzer

Program Lua pertama saya LESGO!

## Cara Menjalankan
1. Install Lua.
2. Jalankan perintah berikut di terminal vcsode :
   ```bash
   lua main.lua
   ```
3. Ketik teks dan tekan enter.

## Contoh Output
```
text analyzer
masukkan teks: saya suka belajar lua

hasil
jumlah karakter : 22
jumlah kata     : 4
kata terpanjang : belajar
```
