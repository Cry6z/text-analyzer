# text analyzer

program Lua pertama saya LESGO!

## jalankan :
1. install Lua.
2. jalankan perintah berikut di terminal vcsode :
   ```bash
   lua main.lua
   ```
3. ketik teks dan tekan enter.

## contoh : 
```
text analyzer
masukkan teks: gibran suka makan etanol

hasil
jumlah karakter : 24
jumlah kata     : 4
kata terpanjang : gibran
```
